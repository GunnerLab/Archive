MCCE

MCCE (Multi-Conformation Continuum Electrostatics) is a biophysics simulation program combining continuum electrostatics and molecular mechanics. In this program, the protein side chain motions are simulated explicitly while the dielectric effect of solvent and bulk protein material is modeled by continuum electrostatics. MCCE is suitable to study:

  1. The protein structural response to the charge;
  2. The charge state response of ionizable residues to structural changes of the protein;
  3. The structural and ionization changes caused by environment pH/eh;
  4. The proton coupled electron transfer;
  5. Side chain conformation prediction.

The mcce program on Hestia is installed at /home/mcce/mcce1.0, and the tool excutables are at /home/mcce/mcce1.0/bin. This path is already in your search path. Here is a summary of these installed programs:

  • mcce: mcce main program
  • getpdb: get pdb file from Protein Data Bank
  • mfe.py: compute ionization energy with mean field approach
  • mk_iatom: build or update atom indecis for mcce parameter file
  • mostocc.py: extract the most occupied conformers from each residue
  • reduce_conf.py: delete unoccupied rotated conformers
  • testfmt: test the DelPhi unformatted pdb file

The mcce web analysis tool can be accessed in campus from:

http://134.74.186.110/cgi-bin/index.py

The most up-to-date mcce main page is:

http://www.sci.ccny.cuny.edu/~jmao/mcce

System administrator should install mcce under /home to make the program accessible by slave nodes.
