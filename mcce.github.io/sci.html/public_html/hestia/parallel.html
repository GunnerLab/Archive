Parallel Programming

Parallel programming on Hestia is supported by MPICH library.

To compile parallel code (cpi.c for example):

mpicc -o cpi cpi.c

To run parallel code:

mpirun -np 8 cpi

There is a book about MPI and parallel programming in Gunner Lab.
