Download and install MCCE 1.0:

Before getting started.  You must obtain a DelPhi site license to use MCCE from the Honig laboratory at Columbia University (http://trantor.bioc.columbia.edu/delphi/).

Please reference Georgescu et. al (2002) and Alexov and Gunner (1997) in any publications that make use of the code (ref 1 and 2 below).

Linux version mcce

This package is mcce main program and tool kit. It can be installed system-wide or in user's directory.

  1. Download the package
    mcce-1.0.tar.gz

  2. Unpack the package
    tar -zxvf mcce-1.0.tar.gz

  3. Run installation script
    cd mcce-1.0
    ./mcce_install

  4. Modify search path $PATH to include mcce bin directory

Linux version web analysis tool

This package is web interfaced mcce analysis tool. It must be installed by super user of the computer.

  1. Download web analysis tool package
    mcce_webtool-1.0.tar.gz
  2. Unpack the package
    tar -zxvf mcce_webtool-1.0.tar.gz

  3. Run installation script
    cd mcce_webtool-1.0
    ./webtool_install
